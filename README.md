## 🚀 Installation & Setup

1. **Clone the repository:**
   `git clone [Link to this Repo]`

2. **Install PHP dependencies:**
   `composer install`

3. **Setup Environment:**
   `cp .env.example .env`
   `php artisan key:generate`

4. **Run Migrations:**
   `php artisan migrate`

5. **Start API Server:**
   `php artisan serve`
